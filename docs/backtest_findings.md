### 3. 多周期共振已集成 ✅

**修改位置**: `src/backtest/engine.py` lines 298-303

**实现**：
```python
if self.multi_cycle is not None:
    mc_result = self.multi_cycle.check_resonance(code, date, lookback_months=6)
    if mc_result.weekly_trend == "down":
```

**过滤逻辑**：知识库"周线定方向"原则
- 月线：用于仓位管理（由共振等级决定）
- 周线：趋势过滤，周线向下禁止做多
- 日线：寻找具体买卖点

### 4. 策略逻辑写死 (架构问题)

**问题**：
- 策略逻辑（入场规则、出场规则）硬编码在 `engine.py`
- `StrategyParams` 只参数化了数值，没有参数化策略逻辑
- 无法切换不同的策略逻辑（如趋势跟踪 vs 突破策略）

### 5. 探索阶段 vs 正式开发

**当前阶段**：探索
- 目标：验证策略思路是否有效
- 允许：快速试错、写死逻辑

**后续转型**：
- 策略定型后，需要把逻辑模块化
- 支持多策略切换和对比
- 支持参数优化

### 关键发现

**波浪理论**：
- 在真实A股数据上返回全UNKNOWN
- 教科书式5浪结构难以在短期数据上形成
- 适合中长期判断，不适合短期日线波段

**缠论**：
- 需要形成明确中枢（3段以上重叠）
- 2024 H1未形成任何中枢
- 适合中长期趋势明显的市场
