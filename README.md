# SwingTrade

个人波段量化交易系统

## 项目概述

专注于A股波段交易的量化投资系统，支持数据采集、策略回测和实盘交易。

## 技术栈

- **数据源**: tushare、akshare、baostock
- **核心库**: pandas、numpy
- **存储**: MySQL + SQLAlchemy
- **分析**: Jupyter、matplotlib、seaborn
